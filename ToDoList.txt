Classificador:
    Extrai FEATURES do training set com todas classes 
    Criar um vocabulario de features by clustering as features (kNN, etc)
    Treina o classificador (com o mesmo ou outro set de imagens)
    -----
    para cada imagem de treino
    tem um FeatureDescriptor, DescriptorExtractor, BOWTrainer -> gera 1 Vocabulario para tudo
    
    para cada imagem de treino
    tem um FeatureDescriptor, DescriptorExtractor, DescriptorMatcher, BOWImgDescriptorExtractor -> gera histogramas pra cada imagem
    
    treinar o classificador com os histogramas e labels das classes
